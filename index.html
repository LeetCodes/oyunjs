Oyunjs

An experiment with nodejs on building a turn-based game server.

Download this project as a .zip file Download this project as a tar.gz file

OyunJS is an experiment going on with node.js to build a turn-based game server, which supports extensions.

Feel free to send comments, flames, blackmail to @ankthepot.

A basic/sample extension (Tic Tac Toe) could be found under server/extensions/tictactoe.js.
